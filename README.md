# CS--350-o-Portfolio-
## Project Summary
Leonard Foy 


For this portfolio, I selected my Morse Code State Machine project and my Thermostat project. Both projects used Python on a Raspberry Pi to control hardware components. The Morse code project solved the problem of sending messages through LED blink patterns using dots, dashes, and pauses. The thermostat project solved the problem of reading temperature input and responding through buttons, LEDs, and display output to simulate a basic embedded thermostat system.

## What I Did Well

I did well with troubleshooting and problem solving. I worked through coding errors, GPIO setup issues, and hardware connection problems. I also did well using state-machine logic to organize how the programs moved between different actions. This made the projects easier to understand and helped the hardware respond correctly to software commands.

## Where I Could Improve

I could improve by planning my code structure earlier and adding comments while I am coding instead of after the project is mostly finished. I could also improve my confidence with reading wiring diagrams and GPIO layouts, especially when the course images do not exactly match the physical setup. More practice with hardware documentation would help me work faster.
